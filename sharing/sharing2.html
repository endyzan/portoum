Download Laragon

  1. Buka Website Resmi Laragon
    • Buka browser (Chrome/Edge).
    • Kunjungi situs resmi: https://laragon.org
  2. Pilih Versi Laragon yang Akan Diunduh
    Biasanya ada dua pilihan:
    • Laragon Full → sudah ada Apache, MySQL/MariaDB, PHP, Node.js.
    • Laragon Lite → versi ringan, tanpa Node.js.

    Untuk pemula dan penggunaan PHP/Laravel, pilih Laragon Full.
    Klik tombol Download pada versi yang kamu pilih.
  3. Tunggu Proses Download
    Ukuran file sekitar 130–140 MB (Full), jadi tunggu hingga file .exe selesai terunduh.
  4. Jalankan Installer Laragon
    • Buka folder Downloads.
    • Klik dua kali file seperti:
    laragon-wamp.exe atau laragon-full.exe.
  5. Mulai Instalasi
    Ikuti langkah berikut:
    • Klik Next.
    • Pilih folder instalasi (boleh default: C:\laragon).
    • Klik Next lagi.

    Centang:
    ✔ Add Laragon to PATH (opsional tapi disarankan)
    ✔ Auto start all services (jika ingin Apache/MySQL otomatis berjalan)

    Lalu klik Install.
  6. Tunggu Proses Instalasi
    Tunggu hingga instalasi selesai (biasanya <1 menit karena Laragon sangat ringan).
  7. Jalankan Laragon
    Setelah selesai:
    • Klik Finish atau buka Laragon dari desktop.

    Ketika Laragon terbuka:
    • Tekan Start All untuk menjalankan Apache & MySQL.
    Jika berjalan normal, status akan berubah menjadi Running.
  8. Cek Laragon Berfungsi
    Buka browser dan ketik:
    http://localhost
    Jika halaman Laragon muncul, berarti berhasil!
  9. (Opsional) Install Laravel dari Laragon
    Laragon sudah punya fitur otomatis.

    Buka Terminal Laragon lalu ketik:
    composer create-project laravel/laravel nama-proyek

    Atau cukup klik kanan di folder www → Quick app → Laravel (jika tersedia).
